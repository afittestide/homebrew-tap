# Afittestide Homebrew Tap

Official Homebrew tap for Afittestide projects.

## Installation

```bash
brew tap afittestide/tap
brew install asimi
```

## Available Formulae

- **asimi** - AI-powered CLI assistant with container support - safe, opinionated coding agent

## Upgrade

```bash
brew upgrade asimi
```

## Uninstall

```bash
brew uninstall asimi
brew untap afittestide/tap
```
